# PROJECT CONTEXT

You are a Senior Software Architect, MERN Stack Engineer, Product Manager, Database Architect, AI Engineer, Security Engineer, UI/UX Designer, and Hackathon Mentor.

Build a production-ready MERN Stack web application called:

# Sondhan AI

An AI-powered Lost & Found Management System designed specifically for Bangladeshi universities such as:
*UU(uttara university)
* BUET
* DU
* CUET
* RUET
* KUET
* SUST
* NSTU
* BRAC University
* AIUB
* NSU
* Jahangirnagar University
* Comilla University
* Islamic University

The system solves the common problem where students lose:

* University ID Cards
* Student Cards
* Mobile Phones
* Wallets
* ATM Cards
* Laptops
* USB Drives
* Headphones
* Keys
* Books
* Lab Equipment
* Calculators

Currently these are shared through:

* Facebook Groups
* Messenger
* WhatsApp Groups
* Department Notice Boards

which causes:

* Scattered information
* Fake claims
* Delayed recovery
* Poor verification
* Lack of tracking

The goal is to provide one centralized platform where students can quickly report, search, verify, claim, and recover lost belongings.

## AI Lost & Found Matching

Using Gemini API

Whenever a Lost Item is created:

Compare against all Found Items.

Factors:

* Title Similarity
* Category Similarity
* Description Similarity
* Location Similarity

Generate:

```js
Match Score
Similarity Percentage
Confidence Level
```

Example:

```json

  "foundItem": "Blue Wallet",
  "similarity": "92%",
  "confidence": "High"
```
